# Compound OS

## Vision

Compound OS is an investment operating system.

Not a portfolio tracker.

Not a stock screener.

It helps long-term investors make better decisions.

---

## Principles

Global index is the base.

Quality before yield.

Investment philosophy drives the application.

Every feature should help users make better decisions.

No unnecessary complexity.

No hype.

No social features.
